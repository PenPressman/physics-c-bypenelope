The good news about this unit is that you’ve already had an intro to almost every concept… just the linear version. You’ll notice that, even for things that don’t just mean “the angular version of [insert something we already learned,” a lot of stuff looks pretty similar. When in doubt, think of the version of things you already know well (ie COE, kinematics, COP etc) and then work on translating it to the rotational version. Also, this unit is fun because you’ll be able to explain a lot of cool real-world phenomena in a technical way. Yay!
