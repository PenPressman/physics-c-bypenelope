Newton’s Laws (coming soon)

1st: Objects stay in the same state of motion (i.e., rest or constant velocity) until acted on by a non-zero force.

2nd: Force = Mass x Acceleration (F = ma)

3rd: For every action/force applied to an object, there is an equal and opposite one back on the applier. (FAB = - FBA)

Formatted Paragraph

In a nutshell, Newton’s laws let us describe and analyze motion. We can derive lots of other stuff from these “first principles.” Also, as you enter this unit, remember not to make any assumptions about what is causing an object to move. I remember this as "there's a source for every force," meaning, if, for example, there is ball hanging from a string, and the string is attached to a moving plane, the string is the only thing pulling the ball up, not the plane. This will all make sense later, but I like to start by thinking about it.

SI Units for Newton’s Laws

Quick note (which you’ll see on every topic page), as long as your units are consistent when you are converting things, use whatever you want, just remember what the base units are.

Also, I’m considering each set of units cumulative, so make sure you know any units from topics before this (i.e. kinematics).

force newtons [N]
spring constant newtons per meter [N/m]
mass kilograms [kg]
newton kilogram meter per second [kgm/s 2]
