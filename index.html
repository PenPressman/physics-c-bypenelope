This website contains resources for AP Physics C: Mechanics and Electricity and Magnetism (E & M), some Thermodynamics, and some Modern Physics. The explanations are written by me, a high schooler, but they are based on what I have learned from 2.5 years of physics classes, multiple textbooks which you can find the names of on the sources page, as well as various resources. For the latter two sections, I am starting with helpful sources since it will take me some time to get everything up-to-date and included. If you'd like more detail about something, reach out!

I'm working on updating this site with more content for other areas of physics (optics, relativity, quantum, other peices of modern physics etc.) so check back soon if you're curious. I've also included some basic intros to concepts if you're in a pre-AP-C physics class.

Use what's helpful, ignore what isn't, and have fun :) If you want to contribute, click the collab button and start a pull request on Github.
